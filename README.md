# 📔 MoodBuddy - Server

## 🖥️ 프로젝트 소개
사용자의 공간에서 쿼디랑 함께 하루의 기록과 고민을 편리하게  작성할 수 있는 일기 웹 서비스입니다 👋🏾👋🏾

## 🕰️ 개발 기간
* 24.03 - 25.03 (약 12개월)
  
## ⚙️ 개발 환경
- `Java 21`
- **IDE** : IntelliJ IDEA
- **Framework** : Springboot(3.3.7)
- **Database** : MySQL
- **ORM** : Hibernate (Spring Data JPA 사용)
